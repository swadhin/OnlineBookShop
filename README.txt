HOW TO COMPILE & RUN :
---------------------
1. RUN ./makes.sh and program will be compiled.
2. "corbaRunCommands.txt" says about how to run the corba application on a single machine.

Main File Descriptions :
-----------------------

1. bookShop.idl (common interface)
2. bookShopClient.java (Client)
3. bookShopServer.java (server)
4. bookShopImpl.java (implementation of server)
5. "bookDatabase.txt" contains the database of the books(Isbn no - Book Name - Author Name - Publisher - Price - Available Qty. - Total Qty.).
